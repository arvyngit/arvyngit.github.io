kita mulai saja ya untuk tutorial setting pppoe nya

untuk setting pppoe kita bisa menggunakan 2 cara yaitu cara manual dan pakai cli/ssh

kita mulai dari cara yang manual terlebih dahulu

setting pppoe manual

  1. login MIKROTIK terlebih dahulu,lalu konfigurasi MIKROTIK agar mendapat internet
  2. Kalau sudah buka menu PPP yang ada di MIKROTIK

    3. dengan posisi komputer telah tersambung ke internet

  3. masuk ke menu pppoe server
  4. tambah pppoe server yang baru dengan menklik tombol +
  5. tambahkan service ke ether 4

    6. Berhubung saya mau menggunakan ether 4 jadi saya pilih ether 4 ,ini terserah kalian mau ditaruh ether berapa

    Jika sudah tekan tombol Apply dan OK

  6. untuk step petama selesai

    7. Jika sudah semua tampilan dari konfigurasi tadi akan seperti ini

    next kita lanjut ke step berikutnya

    SETTING USERNAME DAN PASSWORD

  7. Masuk ke menu SECRETS

    8. lalu tekan tombol + yang warna biru

  8. Add USERNAME dan PASSWORD

    9. kita harus tambahkan USERNAME dan PASSWORD yang akan kita gunakan,pastikan USERNAME dan PASSWORD yang di buat itu aman dan susah untuk di tebak

  9. ADD Local dan Remote address

    10. supaya apa? supaya clinet bisa menggunakan internet dengan PPPOE kita,jadi kita harus tambahkan local dan remot address

    Local Address adalah IP address yang akan ditempelkan ke interface yang ada di PPPoE server.

    Remote Address adalah IP address yang akan ditempelkan ke interface yang ada di PPPoE client.

    Setelah itu click tombol Apply, kemudian OK

    SEKARANG PPPOE SERVER SUDAH BISA DI GUNAKAN

setting pppoe cepat dengan terminal

cara nya mudah dengan masuk ke menu terminal yang ada di MIKROTIK

  1. masuk kemenu terminal
  2. KETIK /ip pool ENTER

    3. ini untuk masuk ke ip pool nya dan memberikan ip

  3. KETIK add name=pppoe ranges=10.0.0.2-10.0.0.5 ENTER

    ini untuk memberikan ip kepada pppoe server nya

  4. kemudian KETIK /ppp profile ENTER

    5. Ini manandakan kita masuk ke menu ppp yang ada di MIKROTIK

  5. KETIK add local-address=10.0.0.1 name=for-pppoe remote-address=pppoe-pool ENTER
  6. KETIK /ppp secret ENTER
  7. KETIK add name=(terserah kalian) password=(terserah) profile=for-pppoe service=pppoe
  8. KETIK /interface pppoe-server server
  9. KETIK add default-profile=for-pppoe disable=no interface=ether4 service-name=pppoeserver

    10. nah kita sudah selesai setting pppoe secara cepat menggunakan terminal yang ada di MIKROTIK

Sekarang kita akan setting pppoe client

  1. BUKA MENU PPP
  2. BUKA MENU interface
  3. TAMBAHKAN INTERFACE BARU
  4. PILIH PPPOE client
  5. TAMBAHKAN PPPoe cient ke ether1

    6. Kita pilih interface menuju PPPoE server kita. Sesuai topologi kita, PPPoE server itu ada di interface ether1.

  6. BUKA MENU Dial-Out
  7. Masukkan Username Dan Password

    8. kemudian tekan tombol APPLY,kemudian OK

  8. SELAMAT KITA SUDAH BERHASIL

    9. Kalau di menu PPP sudah muncul pppoe-out dan disampingnya ada flag R maka settingan PPPoE kita sudah berhasil

    sekarang setting pppoe client pakai terminal

    SELAMAT SETTING PPPOE SUDAH BERHASI

  1. [admin@MikroTik] > interface pppoe-client add interface=ether2 password=StrongPass service-name=pppoeservice name=PPPoE-Out disabled=no user=MT-User
  2. [admin@MikroTik] > interface pppoe-client print
  3. Flags: X - disabled, I - invalid, R - running 0 R name="PPPoE-Out" max-mtu=auto max-mru=auto mrru=disabled interface=ether2 user="MT-User" password="StrongPass" profile=default keepalive-timeout=10 service-name="pppoeservice" ac-name="" add-default-route=no dial-on-demand=no use-peer-dns=no allow=pap,chap,mschap1,mschap2
